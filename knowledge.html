// 什麼是路怒症 (Road rage)？

「路怒症」為駕駛因各種道路狀況，所產生的憤怒、焦躁、緊張等的情緒問題。
輕則出現語言衝突、財損,重則導致肢體衝突甚至危及生命安全。
是不可忽視的心理狀況喔！

// 怎麼處理路怒症？

在行車時盡量放鬆心態,保持心情舒暢，做一些放鬆的事情,像嚼口香糖、放自己喜歡的音樂，來轉移情緒。
或是了解情緒化的原因並避免，如：
‧討厭塞車→提早出門或避開尖峰路段
‧等紅燈不耐煩→聽音樂或廣播轉移注意力
‧他人車品不佳→提醒自己保持冷靜並迴避

// 路怒症的成因？

每個人其實多少都有路怒症，車輛增多以及生活節奏加快，路況很難如想像般順利。
‧他人/自己的車品不佳
‧號誌繁雜/過久
‧三寶的出現
等，會使人心情焦躁，這種激動的情緒下也會導致自己產生過激反應，甚至和其他車輛上的人員發生衝突。
